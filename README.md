# 🪞 Home Dashboard (Magic Mirror PWA)

A modern smart dashboard inspired by smart mirror layouts.

## ✨ Features

- PWA installable app
- Fullscreen kiosk mode
- Snap-grid layout
- Live clock
- Weather API (OpenWeather)
- News API
- Calendar (ICS)
- Day / Night theme
- Offline support (Service Worker)
